# Grade Calculator Project

## Description
A Python program that calculates student grades using:
- if-elif-else conditions
- logical operators (and, or)
- nested conditions
- real-world business rules (attendance requirement)

## Features
✔ Handles invalid marks  
✔ Uses logical flow control  
✔ Displays clear messages  
✔ Easy to read & well-structured  
✔ Tested with multiple inputs
